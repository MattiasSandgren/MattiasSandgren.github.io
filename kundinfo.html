Kundinfo

Tres Platános avtal som gäller finns inte ute offentligt. För att få dessa, kontakta oss på något av de vis som hänvisas på vår sida om kontaktsätt. Det finns möjlighet till att få en provperiod. Man får som företag vara med och bestämma tidsperiod och lösningar med avtal, vid intresse kontakta oss för vidare information. Om man bestämt sig för att man vill bli kund hos oss, kontakta oss så kan vi föra en dialog för att fastställa avtal och speciella bestämmelser gällande frakt, betalinformation m.m.
